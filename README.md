# World Globe Explorer

Premium React + Vite globe explorer with Three.js, `react-globe.gl`, TailwindCSS, Framer Motion, and Lucide icons.

## Run Locally

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Highlights

- Interactive 3D globe with drag, zoom, auto-rotate, and reset camera
- GeoJSON country boundaries with hover and click selection
- Search with fuzzy suggestions
- Distance explorer with animated route arc
- Bookmarks and history persisted in local storage
- Dark glassmorphism UI optimized for Vercel deployment
